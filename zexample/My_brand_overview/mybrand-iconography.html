Sizing & Alignment

Proper alignment ensures the interface feels balanced. Icons should always align optically with the text baseline.

  • Icons should align optically with text
  • Use consistent sizing across interfaces (16px, 20px, 24px)
  • Avoid scaling icons disproportionately

Decoration Only

Redundant Labels
